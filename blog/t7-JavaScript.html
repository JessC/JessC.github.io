Over the past couple months we have been getting familiar with Ruby. So far we've been using two ways to store multiple values to one variable. Those two things are arrays and hashes. Arrays are a bit more simple because there values are always indexed by integers starting from 0. If you were to think of an array as a hash, then the idex number would be called the key and the value stored would be called the value. But the difference between the two is that the key in hashes does not have to be an integer, it can be pretty much anything you choose.
