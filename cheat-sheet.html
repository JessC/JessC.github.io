Sublime Text Cheat Sheet

By: Chris and Jesse

General

Key Press Command
⌘ + N New File
⌘ + S Save File
⌘ + O Open File
⌘ + ⇧ + N New Window
⌘ + ⇧ + W Close Window
⌘ + ⇧ + P Command Palette

Selection/Search

Key Press Command
⌘ + U Undo
⌘ + A Select All
⌘ + D Expand Selection to Word
⌘ + L Expand Selection to Line
⌘ + G Find Next
⌘ + E Use Selection to Find

Moving/Editing

Key Press Command
⌘ + Z, Undo
⌘ + ⇧ + Z Redo
⌘ + Y Repeat
⌘ + ] Indent
⌘ + [ Unindent
⌘ + J Join Lines
