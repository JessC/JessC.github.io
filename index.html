CC Valid

JavaScript | HTML5 | CSS3

CCValid Simply lets you know if a number could be a valid credit card number. Users simple input a number. The number is input into the Luhn algorithm written in Javascript on the front end. A failure alert is output if it doesn't pass. If it passes, a success alert is output including the card type. ex: Visa

Built with JavaScript, jQuery, CSS3, HTML
