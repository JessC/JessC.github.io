CC Valid

JavaScript | HTML5 | CSS3

CCValid Simply lets you know if a number could be a valid credit card number. Users enter a number which is input into the Luhn algorithm. It's written in Javascript on the front end to eliminate sending sensitive information to a database. A failure picture is output if it doesn't pass. If it passes a picture of the card type appears. If it passes but the card type couldn't be determined then a generic card appears.

Built with JavaScript, jQuery, CSS3, HTML
